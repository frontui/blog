1. docco

docco 是 Backbone 作者开源的一个 js 文档生成器。

安装

生成文档

或者自定义生成路径

在同级目录下生成了 docs 文件夹，访问文件里面的 index.html 进行浏览。

文档规范

docco 的文档规范是使用 markdown风格 ,在我们 JS 文件中使用 // 加 空格来使用。

具体可参考 Backbone 源码

文档规范

  1. 头部版权说明

// 加5个 空格

  1. 高亮强调
  1. 标题

大致与 markdown 语法相同。

jsDoc 是一种标准的 js 注释规范，大部分IDE都支持或者其插件。

  1. 规范

    以/**开头以及 **/结尾

    或者多行注释
