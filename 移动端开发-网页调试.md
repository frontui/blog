## 安装weinre

    npm install weinre -g -d

绑定本地IP

    weinre --boundHost 192.168.191.1

获取weinre的js代码

拷贝js代码到页面中


## chrome devtools

安装安卓usb驱动

chrome地址栏输入

    chrome://inspect

手机允许，手机中chrome打开要调试页面

chrome设备tab页面，点击`inspect`
